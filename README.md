# Noti Limun — Portfolio

A single-page immersive digital portfolio for Noti Limun (Ivan Bilinovac) - Rapper, Graphic Designer, and TDKM collective member.

## Features
* **The "Corrupted Grid" Design System**: Brutalist layout combined with glitch-art aesthetics.
* **Cinematic Experience**: Vanilla JS-driven horizontal scrolling inside a vertical scroll container for the discography.
* **Intersection Observers**: Scroll-triggered text reveals and animations calibrated for exact viewport hits.
* **Zero Dependencies**: Built entirely with pure HTML, CSS, and Vanilla JavaScript. No heavy frameworks.
* **Performance Focused**: 60FPS fluid animations using CSS `clip-path` and `transform`, preventing layout thrashing.
